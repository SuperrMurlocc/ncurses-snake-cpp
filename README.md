# ncurses-snake-cpp
### This is a simple snake game project, done in C++ using ncurses lib. 
In order to play the game simply type 
```sh
make && ./winsys
```
in your terminal, after downloading this repo.
In order to clean .o files simply run:
```sh
make clean
```
